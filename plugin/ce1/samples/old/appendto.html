This sample is not maintained anymore. Check out the brand new samples in CKEditor SDK.

The ce1.appendTo() method serves to to place editors inside existing DOM elements. Unlike ce1.replace(), a target container to be replaced is no longer necessary. A new editor instance is inserted directly wherever it is desired. 

ce1.appendTo( 'container_id',
	{ /* Configuration options to be used. */ }
	'Editor content to be used.'
);
